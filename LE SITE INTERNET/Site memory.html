Le jeu du memory 記憶のゲーム (kyoukou no game)

Présentation du jeu

Le Memory est un jeu de cartes proposant un mode de jeu solo.
Le principe est simple : Vous vous retrouvez face à 20 cartes faces cachées, vous allez retourner les cartes 2 à 2 :
si les 2 cartes sont identiques, vous pouvez les retirer du jeu, sinon elles se remettront faces cachées.
L’objectif est de ne plus avoir de cartes devant soi.
Une version du jeu du Memory codée par Tristan ACHARD dans le cadre d'un projet d'Informatique
et de Sciences du Numérique.

  • Le thème du manga japonais exploité pour une ambiance des plus agréables
  • un jeu alliant amusement, reflexion et observation
  • des dessins sophistiqués tirés de fan art de haut niveau
