ISN 2018 : Le jeu du pendu




Le but de ce jeu est de découvrir un mot. Les tirets désignent le nombre de lettres appartenant au mot.


Le joueur annonce une lettre. La lettre fait-elle partie du mot à deviner ?

Si oui, la lettre s'affichera à sa place, sur le tiret correspondant, et autant de fois que la lettre apparait dans le mot

Si non, la première image du pendu apparait.


Le jeu continue jusqu'au moment où l'un des joueurs remportent la partie, c'est-à-dire :


Le joueur remporte la partie en découvrant toutes les lettres du mot

L'ordinateur gagne la partie si l'image du pendu est totalement apparu.


Le cahier des charges


Ce que le joueur peut ou doit faire :


-Choisir une lettre

-Quitter le jeu

-Recommencer une partie


Image du pendu :

- Lorsqu'une lettre n'est pas dans le mot, une image du pendu apparait. Lorsque ce pendu arrive à son terme, le joueur a perdu.

Le joueur peut donc se tromper 8 fois sinon la partie sera terminée et le joueur aura perdu.


Fonctionnement du jeu :


- Choisir les lettres à l’aide de la souris. Une barre avec toutes les lettres sera affichée en haut.

- Lorsque le joueur a gagné une partie, ou alors il l’a perdu, il peut recommencer une partie ou alors quitter le jeu.
